# Article Template

## Metadata Structure
```yaml
---
title: 'Your Article Title'
description: 'A compelling description of your article (150-160 characters)'
author: 'Your Name'
date: 'YYYY-MM-DD'
readingTime: 'XX min'
tags: ['tag1', 'tag2', 'tag3', 'tag4']
series: 'Series Name'
seriesOrder: X
---
```

## Article Components

```jsx
<ArticleHeader

  readingTime="XX min"

```

### Main Content Structure

#### Introduction
- Hook the reader with a compelling opening
- Present the problem or topic
- Establish relevance
- Preview main points

#### Body Sections
- Use clear headings (H2, H3)
- Include practical examples
- Add relevant research
- Provide actionable insights

#### Clinical Insights
- Professional perspective
- Evidence-based information
- Treatment approaches
- Best practices

#### Practical Applications
- Concrete strategies
- Step-by-step guidance
- Real-world examples
- Implementation tips

#### Reflection Questions
- Self-assessment prompts
- Discussion points
- Growth challenges
- Action steps

## Writing Guidelines

### Voice and Tone
- Professional yet accessible
- Compassionate and understanding
- Evidence-based approach
- Trauma-informed language
- Inclusive perspective

### Content Structure
- Clear headings and subheadings
- Short, focused paragraphs
- Engaging examples
- Practical takeaways
- Action steps

### Clinical Accuracy
- Evidence-based information
- Current research citations
- Professional best practices
- Ethical considerations
- Clinical guidelines

### Formatting
- Use markdown for text formatting
- Include relevant images/diagrams
- Add blockquotes for emphasis
- Use lists for clarity
- Include code blocks for structured content

## Submission Process

1. Use this template for your article
2. Follow the style guide
3. Include all metadata
4. Add author information
5. Submit for review

## Review Checklist

- [ ] Metadata complete
- [ ] Structure follows template
- [ ] Content meets guidelines
- [ ] Citations included
- [ ] Images/media added
- [ ] Links verified
- [ ] Tags appropriate
- [ ] Series information correct
- [ ] Author bio updated
- [ ] Related articles identified
